Options
^^^^^^^

The "Options" tab contains additional, minor settings. For now it
contains only a debug flag. This is the same as the flag from the
extension configuration, but for the current content element only.
This makes it easier, for example, to find information in the devlog,
instead of having all Display Controllers produce debug output.
